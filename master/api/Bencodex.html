Namespace Bencodex

Classes

Codec

The most basic and the lowest-level interface to encode and decode Bencodex values. This provides two types of input and output: Byte arrays and I/O Streams.

CodecException

Serves as the base class for codec exceptions.

DecodingException

The exception that is thrown when an input is not a valid Bencodex encoding so that a decoder cannot parse it.
