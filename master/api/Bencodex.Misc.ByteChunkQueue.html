Class ByteChunkQueue

A special-purpose queue for internal use, which accepts inserting a chunk of Bytes into and removing the number of Bytes from.

For example, if foo, bar, and baz were Pushed into a queue, the first Pop(5) returns fooba and the second Pop(5) returns rbaz.
