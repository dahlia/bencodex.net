Dictionary(IEnumerable<KeyValuePair<IKey, IValue>>)

Converts an any IEnumerable<T> of IKey--IValue pairs to a corresponding Bencodex Dictionary value.

Declaration
public Dictionary(IEnumerable<KeyValuePair<IKey, IValue>> value = null)
Parameters
Type Name Description
IEnumerable<KeyValuePair<IKey, IValue>> value

An any IEnumerable<T> of KeyValuePair<TKey,TValue> elements to convert. As the contents of this object is copied, changing this object does not affect the contents of created Dictionary. Be default, null is passed, which creates an empty Dictionary.
